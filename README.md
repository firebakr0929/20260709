# 臺灣地震測報與次世代科技應用簡報與互動平台

本儲存庫收集了由 **中央氣象署 地震測報中心 陳達毅科長** 所撰寫與發表的地震防災宣導與次世代科技應用簡報（Markdown 格式），並內建一個**高度互動的網頁展示平台**。

## 🌐 互動式網頁功能特色
本儲存庫已整合一個精美的單頁應用程式（SPA），具備以下核心功能：
1. **📊 簡報播放器 (Slide Player)**：直接解析本儲存庫中的原始 Markdown 投影片，具備鍵盤控制、自動播放、全螢幕展演、以及**語音朗讀 (Text-to-Speech)** 功能。
2. **🌍 地震預警物理模擬器 (EEW Simulator)**：以動畫展示 P波 與 S波 在臺灣島上的傳播速度差，動態運算各縣市在「現地型預警」與「區域型預警」下的強震倒數、避難窗口及「預警盲區」。
3. **🤖 AI 代理群協同應變流程 (AI Agents Network)**：模擬未來氣象署導入 AI 代理（數據整合、快速定位、震度圖繪製、災情廣播、LLM 報告產出與市民問答）時的協作工作流與日誌。

---

## 🛠️ 開發與本地執行

本專案採用純前端技術（HTML5, Vanilla CSS, Vanilla JS），無須安裝任何繁雜的依賴或編譯：
1. 本地預覽：使用任何本地網頁伺服器（例如 VS Code 的 Live Server 或是 Python `python -m http.server 8000`）在本地開啟 `index.html`。
2. 以 Node.js 執行：
   ```bash
   npx http-server . -p 8000
   ```
   然後在瀏覽器打開 `http://localhost:8000` 即可預覽。

---

## 🚀 部署至 GitHub Pages

本專案已配置 GitHub Actions 工作流。當您將代碼推送到 `main` 分支時，系統會自動啟動部署並發布至 GitHub Pages。

**部署步驟：**
1. 確保已於 GitHub Repository 設定中開啟 GitHub Pages 選項（選擇使用 GitHub Actions 部署）。
2. 推送至 GitHub：
   ```bash
   git push origin main
   ```
   推送成功後，GitHub Actions 會自動建置並部署，您可在 Repository 頁面的 `Actions` 頁籤或設定頁中找到網頁的公開網址（例如：`https://<您的帳號>.github.io/20260709`）。
